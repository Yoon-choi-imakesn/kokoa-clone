# kokoa-clone
Kokoa Clone with Nomadcoders started 02.01.2022.

## 학습 계획
1. clone 이전 이론 
    이론은 빠르게 복습. 지난 이론 공부에서 배우지 않았던 부분 추가 정리. 
2. clone
   1) 주어진 페이지 스스로 만들어보기
        - 이후 수업에서 나올 내용 (구조, 변수명, 방법) 달라지더라도 신경쓰지 않고, 우선 혼자 publishing하기
        - 혼자 할 수 없는 부분이 있다면 기록해놓기
   2) 영상 따라가면서
        - 나와 다른 부분을 어떻게 했나
        - 막혔던 부분 solution 방법

    3) 복습
        a. 해당 파트 공부한 당일에 혼자 다시 복습
